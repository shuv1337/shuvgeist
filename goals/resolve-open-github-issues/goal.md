# Resolve Open GitHub Issues

Resolve every GitHub issue that is open in `shuv1337/shuvgeist` when execution begins. Work implementation-ready Electron issues first, handle human-decision issues through researched recommendations, and finish with each issue either closed or explicitly documented as blocked with the next human action.

Use `goals/resolve-open-github-issues/facts.md` as the shared understanding of scope and done criteria.

Use `goals/resolve-open-github-issues/plan.md` as the approved execution plan.

Done condition: all open issues are either closed with validated fixes/decisions pushed to GitHub, or explicitly documented as still blocked with the current blocker and next human action.
